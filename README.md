# Board Manager 🎲

보드게임 동아리를 위한 종합 관리 시스템

## 📋 프로젝트 개요

Board Manager는 보드게임 동아리의 효율적인 운영을 위한 데스크톱 애플리케이션입니다. 회원 관리, 활동 일정, 회비 관리 등 동아리 운영에 필요한 모든 기능을 하나의 시스템에서 제공합니다.

## ✨ 주요 기능

### 🔐 인증 및 권한 관리
- **로그인 시스템**: 아이디/비밀번호 기반 인증
- **권한 계층**:
  - 👨‍🏫 관리자(담당 교수): 모든 권한 보유
  - 👑 부장: 전체 기능 사용 가능
  - 👥 평회원: 제한된 기능 사용

### 👥 회원 관리
- 신규 회원 가입 (아이디 중복 체크)
- 회원 정보 조회 및 수정
- 회원 추가/삭제 (부장, 관리자만 가능)
- 직책 부여 및 관리
- 회원 정보: 아이디, 학번, 이름, 성별, 생년월일

### 📅 일정 관리
- **스케줄 등록**:
  - 날짜, 장소, 참가 인원 설정
  - 활동 내용 기록
  - 인원 목록에서 참가자 선택
- **진행 상태 관리**:
  - 예정된 스케줄 목록 조회
  - 체크박스를 통한 완료 처리
  - 완료된 일정은 자동으로 활동 내역으로 이동

### 📝 활동 내역
- 완료된 활동 기록 자동 저장
- 활동 정보: 날짜, 장소, 참가 인원, 활동 내용
- 과거 활동 이력 조회

### 💰 회비 관리
- 월별 회비 자동 계산 (인원 × 10,000원)
- 지출 내역 기록:
  - 날짜, 장소, 사용 대표자
  - 사용 목적 및 금액
  - 잔여 회비 자동 계산
- 회비 사용 내역 조회

## 🛠 기술 스택

### Backend
- **Language**: Java 21
- **Framework**: Spring Boot 3.2
- **Database**: MySQL

### Frontend
- **GUI Framework**: Java Swing
- **Layout**: NetBeans GUI Builder

### Architecture
```
src/
├── controller/     # 컨트롤러 계층
├── service/        # 비즈니스 로직 계층
├── repository/     # 데이터 접근 계층
├── domain/         # 도메인 모델 (Entity)
├── dto/            # 데이터 전송 객체
├── exception/      # 예외 처리
└── view/           # GUI (JFrame, JPanel)
```

## 🎯 사용 방법

### 1. 초기 로그인
- 더미 데이터로 생성된 관리자 계정으로 로그인
- 관리자 계정 정보는 초기 설정 시 제공

### 2. 회원 가입
- 로그인 화면 하단의 "회원가입" 링크 클릭
- 필수 정보 입력 및 아이디 중복 확인
- 가입 완료 후 승인 대기 (관리자 승인 필요)

### 3. 스케줄 등록
- 스케줄 탭에서 새 활동 계획 작성
- 참가 인원을 목록에서 선택
- 등록 버튼으로 스케줄 추가

### 4. 활동 완료 처리
- 스케줄 목록에서 완료된 활동 체크
- "수행 완료" 버튼 클릭
- 자동으로 활동 내역으로 이동

### 5. 회비 관리
- 회비 관리 탭에서 지출 내역 등록
- 잔여 회비 실시간 확인

## 📱 주요 화면

### 로그인 화면
- 아이디/비밀번호 입력
- 로그인 시도 제한 (보안)
- 회원가입 및 계정 찾기 링크

### 메인 대시보드
- 탭 기반 네비게이션
- 권한별 메뉴 접근 제어
- 인원 관리, 활동 내역, 스케줄, 회비 관리

## 🔒 보안

- 비밀번호 암호화 저장
- 권한 기반 접근 제어 (RBAC)
- 세션 관리
- SQL Injection 방지
